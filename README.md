# zvij.si

Static site rebuild (Astro) for migrating the current WordPress site to custom HTML while preserving look & feel.

## Structure

- `src/` – source code (Astro components, pages, content)
- `public/` – static assets
- `dist/` – build output
- `docs/` – migration notes & discovery artifacts

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy

Deploy the `dist/` folder to your static hosting provider.
