# Zvij.si Design Direction

## Current Direction

The dev shop prototype should feel:

- adult
- calm
- clean
- slightly underground without looking childish
- warm, paper-like, and ritual-focused
- more like a trusted setup companion than a generic webshop

Primary line:

```text
Tvoj ritual. Tvoja mera. Tvoj setup.
```

## Visual System

The first theme pass uses:

- warm beige / paper background
- dark ink-like typography
- muted green accent
- restrained warm brown accent
- strong headline scale
- compact cards with 6px radius
- product cards that prioritize category, title, and simple action
- mobile-first stacking for hero, cards, and shop grid

## Homepage Structure

The homepage now includes:

1. Hero with primary message and two CTAs.
2. Product pillars:
   - DUBI filtri
   - CBD čaj
   - Zvij setup
   - Reload
3. `Član Zvij.si` section for future Zvij koda / dobroimetje / repeat-order logic.
4. Starter setup section around DUBI filtri, rolca, and reload logic.
5. CBD čaj section with careful tea-only copy.
6. Footer CTA: `Začni s svojim setupom.`

## Copy Guardrails

CBD copy must stay around:

- CBD čaj
- konopljini vršički za čaj
- ritual
- mirnejši ritual
- brez THC učinka

Avoid:

- smoking/inhalation language
- medical claims
- guaranteed effects
- disease, therapy, pain, anxiety, sleep, or cure claims

Membership copy must avoid:

- resale framing
- dealer language
- MLM language
- commission or cash payout language
