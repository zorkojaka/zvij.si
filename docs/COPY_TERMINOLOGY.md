# Copy Terminology

## Correct

- CBD vršički
- CBG vršički
- CBD/CBG vršički
- premium vršički
- izbrani vršički
- čajna uporaba
- za pripravo čaja
- lahko se uporablja kot čaj
- brez THC učinka
- ritual
- jasna mera
- reload
- reload paket
- reload opomnik
- naslednji reload

Use `laboratorijsko analizirano` only if source analysis exists and is linked or otherwise verifiable.

## Avoid

- CBD čaj as the main product name
- CBD čaj as the main category/page identity
- smoking instructions
- medical claims
- anxiety, sleep, pain, stress relief, treatment, illness, or similar
- legal high
- HHC as an active offer
- refill as the main public term
- refill paket / refill sistem / refill opomnik
- language implying customers return empty packaging for refilling
